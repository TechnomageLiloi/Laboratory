# Counter

## Task

Set number increasing and showing at 7 segment display.
Write library class for this.

Note: in this project library class robustly set to pins (see. logic scheme)

## Components list

* Arduino Uno
* 1 x 7 segment display
* 8 x 1kOhm
* 8 x wire
